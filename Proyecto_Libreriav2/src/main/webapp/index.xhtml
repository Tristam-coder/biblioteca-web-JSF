Inicio - Sistema de Biblioteca

Bienvenido al Sistema de Biblioteca

Gestión integral de biblioteca con tecnología Jakarta EE

Sistema de Gestión Bibliotecaria

Plataforma completa para la administración de bibliotecas que integra gestión de usuarios, catálogo de obras, préstamos, reservas y más.

Desarrollado con Jakarta EE 10, JSF, BootFaces y RESTful APIs.

Módulos del Sistema

Administra los usuarios del sistema: socios, bibliotecarios y administradores.

  • Crear y editar usuarios
  • Gestionar permisos
  • Consultar historial

Gestiona el catálogo completo de libros, revistas y material audiovisual.

  • Registrar nuevas obras
  • Gestionar ejemplares
  • Clasificar por categorías

Controla los préstamos activos, devoluciones y renovaciones.

  • Registrar préstamos
  • Procesar devoluciones
  • Calcular multas automáticas

Gestiona las reservas de obras que están prestadas.

  • Crear reservas
  • Notificar disponibilidad
  • Gestionar cola de espera

Administra las multas por retrasos en devoluciones.

  • Registrar multas
  • Procesar pagos
  • Consultar historial

Consulta el historial de operaciones y estadísticas.

  • Historial de préstamos
  • Auditoría de cambios
  • Notificaciones del sistema

API REST Disponible

Este sistema cuenta con una API RESTful completa que permite integración con otras aplicaciones y clientes externos.

URL Base: #{request.contextPath}/api

Endpoints Disponibles:

  • GET /api/usuarios - Listar usuarios
  • GET /api/obras - Listar obras
  • GET /api/prestamos - Listar préstamos
  • POST /api/usuarios - Crear usuario
  • ... y muchos más
